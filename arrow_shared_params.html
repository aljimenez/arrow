Sharing Params in Descriptor

To share parameters across tests in the same descriptor,you can define them like this 


            "config": {

                "descriptorSharedParams" :{
                    "yhooquote": "Yahoo! Inc. (YHOO)",
                    "applequote": "Apple Inc. (AAPL)"
                }
            }

Then, the parameters in "descriptorSharedParams" can be accessed across all tests in the descriptor. 
            [
                {
                    "settings": [ "master" ],

                    "name": "controllers",

                    "config": {

                        "descriptorSharedParams" :{
                            "yhooquote": "Yahoo! Inc. (YHOO)",
                            "applequote": "Apple Inc. (AAPL)"
                        }
                    },

                    "dataprovider" : {

                        "Test YHOO Ticker" : {
                            "group" : "func",

                            "params" :{

                                "scenario": [
                                    {
                                        "page": "http://finance.yahoo.com"
                                    },
                                    {
                                        "controller": "locator",
                                        "params": {
                                            "value": "#txtQuotes",
                                            "text": "yhoo\n"
                                        }
                                    },
                                    {
                                        "test": "test-quote-yhoo-shared-params.js"
                                    },

                                    {
                                        "page": "http://finance.yahoo.com"
                                    },
                                    {
                                        "controller": "locator",
                                        "params": {
                                            "value": "#txtQuotes",
                                            "text": "aapl\n"
                                        }
                                    },
                                    {
                                        "test": "test-quote-aapl-shared-params.js"
                                    }

                                ]
                            }

                        }
                    }
                },
                {
                    "settings": [ "environment:development" ]
                }

            ]

Example of descriptor with shared params From your test file, you can access these shared params as 
var quote = this.testParams.descriptorSharedParams["applequote"];
